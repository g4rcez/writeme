## Github

You can post any github link with [code-syntax](/features#code-syntax) with `ogp` as language of code block.
That's it. Only.

```ogp
https://github.com/octocat/Hello-World
```

## Youtube

Another OpenGraph support is Youtube videos. Just paste video link :D

```ogp
https://www.youtube.com/watch?v=rt-2cxAiPJk
```
