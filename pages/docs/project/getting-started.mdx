# Install

For alpha release, install is not available, please, use

```bash
yarn create next-app my-docs --typescript --example https://github.com/g4rcez/writeme
```

This command will do the default install of NextJS and add the necessary files for the initial configuration. After this, run the `init` command to exclude all blog/docs in project and start to write yours

```bash
cd my-docs
yarn run init
```

# Write Docs :book:

Every `.md` or `.mdx` will be converted to document route. For SSR purpose and file management, you must write the header in [front-matter](https://jekyllrb.com/docs/front-matter/) format. Writeme use this format:

```yaml
---
title: Page Title
description: Simple description
project: Project name, to group purpose
repository: Github repository
order: Order in project section
sidebar: Parent order in sidebar
tags: ["writeme", "tutorial"] # Tags for subjects
---
```

project
: Project in front-matter is your "group" for documents 

order
: Order represent the numeric position in sidebar section children

sidebar
: Represent the numeric position in sidebar titles
