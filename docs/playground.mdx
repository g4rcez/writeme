This component playground use [Sandpack](https://sandpack.codesandbox.io/) an awesome Playground powered by [Codesandbox](https://codesandbox.io/).

# How can I use it?

To include and playground at your document, just use the JSX syntax `<Playground />`

<Playground />

Or you can use the markdown syntax

```
```playground
\```
```

```playground
```

The default template of playground is `react-ts`

# Known issues

- Templates like vue/vue3 are not work by default. We need to change the template and the environment to work together.
- You cannot pass multiple files to one playground
